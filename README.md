**Generic OMS Framework**

**Overview**

Generic Order Management System (OMS) Framework designed using Microservice Architecture.

The framework follows a 60/40 model:

- 60% Reusable Core Services
- 40% Customizable Business Modules

**Technology Stack**

- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- JWT
- Docker
- Docker Compose

**Core Services**

- Auth Service
- User Service
- Order Service
- Payment Service
- Notification Service
- Audit Service

**Project Status**

 Architecture & Security Design Phase

**Goals**

- Reusability
- Security
- Scalability
- Maintainability
- Domain Customization

**Supported Domains**

- E-Commerce
- Education
- Healthcare
- Manufacturing
- SaaS
